                                                       MYSQL ALL QUERIES 

1)Listing All Existed Databases:-

SHOW DATABASES;

2)Create Database:-

CREATE DATABASE <DB_NAME>;

3)Shift Database to Current Create Database:-

USE <DB_NAME>;

4)Clear All Cmd:-

system cls;

5)Cmd To Check Which Database you Are Using:- 

SELECT DATABASE();

6)DELETING A DATABASE:-

DROP DATABASE<DB_NAME>;

7)CREATE A COLLECTION IN DATABASES:-

CREATE TABLE <TB_NAME>(<COL1> <DATATYPE>,<COL2> <DATATYPE>);

8)Check Table is Created Or Not:-

SHOW TABLES;

8)DELETING TABLE FROM DATABASE:-

DROP TABLE <TB_NAME>;

9)Cmd for Table Information:-

18) Default Value Change in Table:-

19) Update Multiple Entity Corresponding to single Attribute:-

UPDATE CUSTOMERS
SET DIVISION = CASE 
    WHEN ID = 101 THEN '8A11'
    WHEN ID = 102 THEN '8B12'
    WHEN ID = 103 THEN '8C13'
    WHEN ID = 104 THEN '8A6'
    ELSE DIVISION -- Keeps the value unchanged for other rows
END
WHERE ID IN (101, 102, 103, 104);

20) Rename Table Name Or Attribute Name:-

ALTER TABLE <TB_NAME> RENAME TO <NEW_TABLE_NAME>

ALTER TABLE <TB_NAME> RENAME COLUMN <OLD_COL_NAME> TO <NEW_COL_NAME>;

21) Check If Any Primary Key Exist:-

             SHOW INDEX FROM <TB_NAME>;

Modify the Column and Add New Primary Key with auto increment:-

            ALTER TABLE <TB_NAME>  MODIFY COLUMN <COL_NAME> INT PRIMARY KEY     AUTO_INCREMENT;
